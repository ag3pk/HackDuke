Twilio Client for Android HowTo's

We've provided a few complete example apps to give you an idea of what you can do with the SDK.

However, each app requires a publicly-accessible web server that Twilio's services can contact to set up calls. Because of this, there are a few things you'll need to do to get the apps running.

BasicPhone

BasicPhone is an application that can accept an incoming call from someone dialing from a regular phone. It can also dial out, but you'll only hear a pre-recorded message.

Stuff you need to do to get it running:

  1. Copy the stuff in the server subfolder to a publicly-accessible web server.
  2. Create a new Twilo Application and set the VoiceUrl to the URL on your server where you put basic-call.php.
  3. In auth.php, replace $accountSid and $authToken with your Twilio account credentials, and replace $appSid with the SID for the application you created above.
  4. Purchase a phone number (or use the Twilio Sandbox Number) and set the VoiceUrl to the URL on your server where you put basic-call.php.
  5. In the Andrdoid application itself, open BasicPhone.java, and edit the constant AUTH_PHP_SCRIPT to point to your auth.php URL.

After you build an install the app on your Android device, you can call the number you purchased (or the Sandbox Number if you used that instead), and you'll be connected to BasicPhone!

ConferencePhone

ConferencePhone lets you create a multi-caller conference call using the <Conference> TwiML verb. You can create a new named conference, join an existing conference, or select people to add to the conference, whether they're also using ConferencePhone or a regular phone.

Stuff you need to do to get it running:

  1. Copy the stuff in the server subfolder to a publicly-accessible web server.
  2. Create a new Twilo Application and set the VoiceUrl to the URL on your server where you put join-conference.php.
  3. In auth.php, replace $accountSid and $authToken with your Twilio account credentials, and replace $appSid with the SID for the application you created above.
  4. In make-call.php, replace $twilioPhoneNumber with a number you've verified with Twilio. Also replace $accountSid and $authToken as before. Finally, change $url to point to where your join-conference.php script lives on your server.
  5. In the Andrdoid application itself, open ConferencePhone.java, and edit the constants AUTH_PHP_SCRIPT to point to your auth.php URL, and MAKE_CALL_PHP_SCRIPT to point to your make-call.php URL.

Now you can build and install the Android app on your Android device and start conferencing!
